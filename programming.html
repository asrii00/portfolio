Multi-Functional Clock

Arduino project (C)

The LCD screen displays the time (kept by an RTC module) and current room temperature (measured by a temperature sensor). The menus can be operated with buttons and an alarm set at a chosen time; at the set time, the clock plays an 8-bit version of Fur Elise.

Group project. My main responsibility was programming the UI displayed on the screen.

Simulated Banking System

Qt / C++

A simulated version of an ATM complete with a card reader assisted login (RFID reader pictured), a touch screen compatible UI, and a database connection. The Qt software package, including Qt Widgets and Qt Designer, was used for development. The user can log in, choose a credit or debit card linked to a dual card (if present), view the account balance, change the PIN code, "withdraw cash", and log out.

Group project. I was the project manager and created the UI; others were responsible for the backend and the card reader functionality.

Movie App

React / TypeScript / Node.js

A fullstack web application for film lovers. You can search for movies and TV series, read and add movie reviews, create and join groups, share news on the group pages, and personalize your profile. The app also has a dark and light mode.

Group project. I was the project manager and worked on both the backend and frontend. The search page, review section, group page and group browsing page were mostly my work, and I created the endpoints required for the group page on the backend side.

Visit the site or the GitHub repository.

News App

Kotlin

A mobile application that offers a simple, no-nonsense way to stay up to date with what's happening all over the world. The app is location agnostic and can display news from any country and in any language that the API (newsdata.io) offers. It also includes functionality to filter and search news, save specific articles to 'Favorites' and adjust the appearance of the app.

Group project. View the GitHub repository.
